## How to run the project

* Clone the repo 
```bash
git clone https://github.com/MagdielPm/Rest_Market.git
```
* Install the dependencies 
```bash
yarn install
```
* Build the project 
```bash
yarn build
```
* Run the project 
```bash
yarn start
```
* Run the tests
```bash
yarn test
```
